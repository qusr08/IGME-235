Description / How To Play

In this very original game, you will be given 1 minute to answer as many questions as you can.

You can select which trivia categories you want as well as the difficulty of the questions below. Choose "General Knowledge" for a variety of topics.

If you get a question right, you are rewarded points as well as time back to the timer. The faster you answer the question, the more points you get. If you get a question wrong, you do not gain any points and lose time from the timer.

You will get more points per question as well as gain or lose more time depending on the difficulty you set. So, the harder the difficulty, the more points you get for a correct question, but the more time you lose if you get a question wrong.

Your goal is to get as many points as possible!

Documentation / Process

As soon as I saw the Open Trivia API, I knew that I wanted to do a trivia game. I'm a big fan of trivia and it was fun to be able to make a small trivia game.

I had a lot of plans for this, with animations and maybe some power-up features, but I definitely got a bit carried away with the ideas. I didn't end up implementing everything I wanted to (maybe I will in the future), but what I have right now I am proud of. I spent a lot of time making sure the UI looked nice and making sure everything worked (which took a lot of hours). I know the code isn't the best and could 100% be optimized further, but it's good enough :).

I didn't use any tutorials or external resources (besides online forums for some coding help). I started by making the trivia game get questions from the API without any UI, and only using the console. At the time, even after the exercise on accessing external sites, I still wasn't sure exactly how it worked. I did a lot of testing and ultimately settled on the code I have now. I'm planning to go all out on Project 3, and I know that the knowledge I gained from this project will help me out.
